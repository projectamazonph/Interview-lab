# Roadmap

## Delivery principle

Deliver risk-reducing vertical slices. Do not complete every layer horizontally before proving an end-to-end workflow.

## Phase 0: Foundation (Completed)

- [x] Auth system (JWT + HttpOnly cookies)
- [x] Dashboard with progress tracking
- [x] Onboarding diagnostic
- [x] Question bank (264+ questions across 6 VA roles)
- [x] Practice answer submission
- [x] AI feedback integration
- [x] Resume upload/review
- [x] Cover letter generator
- [x] Download library
- [x] Admin question manager

## Phase 1: Core vertical slice (In Progress)

- [x] User registration → Onboarding → Role selection
- [x] Mock interview session flow
- [x] AI scoring with rubric breakdown
- [x] Session summary and transcript storage

**Exit condition:** A user can complete a full mock interview from registration to results.

## Phase 2: Workflow completeness

- [ ] Resume DOCX export
- [ ] Certificate generation
- [ ] AI-generated personalized study plan
- [ ] Employer-style timed tests
- [ ] Portfolio builder
- [ ] Community leaderboard

## Phase 3: Monetization

- [ ] Stripe subscription integration (Free/Starter/Pro tiers)
- [ ] Tier-based feature gating
- [ ] Payment history tracking
- [ ] Referral system

## Phase 4: Advanced features

- [ ] Voice interview mode
- [ ] Video response review
- [ ] Job description tailoring
- [ ] Community examples
