# Amazon VA Interview Lab - Scoring Rubrics

## 14.1 Interview Answer Rubric

| Score | Meaning |
|-------|---------|
| 1-2 | Incorrect, vague, or risky |
| 3-4 | Some understanding but lacks clarity |
| 5-6 | Acceptable beginner answer |
| 7-8 | Strong, specific, role-relevant |
| 9-10 | Excellent, practical, confident, and truthful |

### Dimensions:
| Dimension | Weight |
|-----------|--------|
| Technical accuracy | 30% |
| Role relevance | 20% |
| Specificity | 15% |
| Communication clarity | 15% |
| Judgment/risk awareness | 10% |
| Truthfulness | 10% |
