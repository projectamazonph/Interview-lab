# Amazon VA Interview Lab - Product Modules

## 3.1 User Onboarding Diagnostic

The app should begin with a diagnostic quiz.

### Inputs
| Input | Example |
|-------|---------|
| Target role | Amazon PPC VA, Amazon Account VA, Listing VA |
| Experience level | Beginner, intermediate, advanced |
| Tools used | Seller Central, Helium 10, Data Dive, Adbrew, ClickUp, Excel |
| Resume status | No resume, generic VA resume, Amazon-specific resume |
| Interview timeline | Tomorrow, this week, this month |
| Confidence level | Low, medium, high |
| Weakest area | PPC, reporting, client communication, cover letter, Excel |

### Output
A personalized prep plan:
> "You are targeting Amazon PPC VA roles and have general VA experience but limited PPC experience. Your 7-day plan should focus on Amazon Ads basics, ACoS/ROAS, search term reports, campaign naming, and client reporting."

---

## 3.2 Role-based Learning Paths

### Path A: Amazon PPC VA
Core modules:
1. Amazon Ads basics
2. Sponsored Products vs Sponsored Brands vs Sponsored Display
3. Campaign types and campaign taxonomy
4. Keyword research basics
5. Search term report analysis
6. ACoS, ROAS, CTR, CVR, CPC
7. Keyword harvesting and negation
8. Bid and budget adjustment logic
9. PPC reporting and client updates
10. Interview case studies

This path should closely align with PPC operations knowledge base: launch, optimize, scale, maintain stages; campaign launch priorities; keyword research; bid optimization; and reporting cadence.

### Path B: Amazon Account VA
Core modules:
1. Seller Central navigation
2. Listing monitoring
3. Inventory and suppression checks
4. Case log management
5. Order and customer issue escalation
6. Basic reporting
7. SOP discipline
8. Client communication

### Path C: Amazon Listing VA
Core modules:
1. Keyword indexing basics
2. Listing title, bullets, description, A+ content
3. Image and video readiness
4. Review/rating awareness
5. Competitive listing checks
6. Listing optimization interview questions

Listing readiness criteria — images, title/bullets, A+ content, review threshold, pricing, promotions, and inventory — can become a strong training checklist.

### Path D: Amazon Reporting VA
Core modules:
1. Key PPC metrics
2. Reading Sponsored Products reports
3. Reading Sponsored Brands reports
4. Search term report summaries
5. Weekly reporting format
6. Changelog discipline
7. Client-facing insights
